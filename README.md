# taniwhy/chat-gpt-discord-bot

## Requirement

- Makefile
- Docker 20.10.0

## Usage

1. Rename `.env.example` to `.env` and set the value
    - `DISCORD_BOT_TOKEN`
    - `CHATGPT_API_TOKEN`
2. Execute `make all` to start bot.
